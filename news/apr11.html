April 11th, 2018 - The END is now enabled on the server!

The only access to The End is through an End portal. Once all The End portals have been discovered and filled, a public end portal will be set up in spawn for all. The End Dragon will also be respawned on a regular basis.
This means the Elytra is no longer craftable, and End stone has been removed from the server shop.
